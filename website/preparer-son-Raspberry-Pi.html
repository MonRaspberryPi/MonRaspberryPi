Préparer son Raspberry Pi

condition requise

Pour configurer votre Raspberry Pi vous aurez besoin:

  • d'une carte SD: taille minimal de 4Gb et de classe 4 (plus la classe est élevé, plus la carte va vite et donc plus le système va vite).
  • d'un câble HDMI ou RCA vidéo lead (pour les vielles télévisions)
  • d'un clavier et d'une souris
  • d'un câble Ethernet (internet) [optionnel]
  • d'une alimentation micro-USB (en gros un chargeur de smartphone)
  • enceinte [optionnel]

Préparation de la carte SD

formatage

Le but des prochaines manipulations vont être de "nettoyer" totalement de carte SD (la formater avec le logiciel par défaut de Windows n'est pas une bonne solution).

  • Windows:
    • Téléchargez SD Formatter 4.0 for Windows
    • Installez le et exécutez le
    • mettre à "ON" dans les options à "format size adjustment"
    • formatez
  • MAC:
    • Téléchargez SD Formatter 4.0 for Mac
    • Installez le et exécutez le
    • sélectionnez "overwrite format"
    • formatez
  • LINUX:
    • installez gparted (sudo apt-get install gparted)
    • supprimez toute les partitions
    • créer une partition en FAT faisant toute la carte SD

New Out Of Box Software (NOOBS)

NOOBS permet d'installer facilement un OS (c'est un peu le BIOS du Raspberry Pi)
il y a 2 versions:

  • network install only
  • offline and network install
la network install only est vraiment beaucoup plus petite mais ne permet que de faire des installation en étant connecté internet.
la seconde elle est évidement la plus complète puisqu'elle intègre déjà un OS dans le cas ou le Raspberry Pi n'est pas connecter à internet (pour le modèle A ou autre)
Je vous recommande NOOBS "offline and network install" bien évidemment.
téléchargez NOOBS
Il ne vous reste plus qu'a extraire l'archive sur la carte SD
Ça y est votre carte SD est prête!!!
